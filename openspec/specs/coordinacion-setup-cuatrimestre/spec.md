## Requirements
### Requirement: Setup de Cuatrimestre
El sistema SHALL unificar la creación de cohortes, fechas académicas y carga de programas en un asistente paso a paso.

#### Scenario: Setup exitoso
- **WHEN** el coordinador completa todos los pasos del asistente (Cohorte -> Calendario -> Programa -> Padrón inicial).
- **THEN** la UI muestra un resumen final y aprueba la operación, generando en bloque toda la estructura base para la materia en el nuevo ciclo.
