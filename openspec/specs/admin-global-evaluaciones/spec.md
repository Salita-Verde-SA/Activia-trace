## Requirements
### Requirement: Panel administrativo global de evaluaciones
El sistema SHALL brindar una vista unificada a nivel global donde la administración institucional pueda listar y buscar todas las evaluaciones cargadas.

#### Scenario: Consulta global
- **WHEN** un rol de administrador o coordinación accede a la lista global de evaluaciones
- **THEN** el sistema retorna la nómina consolidada de evaluaciones filtrable por materia, cohorte, fecha y estado, permitiendo un control institucional centralizado.
