## Requirements
### Requirement: Generación de bloque HTML para Moodle
El sistema SHALL permitir la generación de código HTML limpio y estructurado que enumere las instancias de encuentro vigentes para su exportación o copia manual al LMS.

#### Scenario: Exportación exitosa de bloque HTML
- **WHEN** el usuario hace click en generar bloque HTML para un conjunto de encuentros
- **THEN** el sistema devuelve una estructura HTML visualmente ordenada que incluye fechas, horarios y enlaces de acceso, lista para ser insertada en la plataforma LMS.
